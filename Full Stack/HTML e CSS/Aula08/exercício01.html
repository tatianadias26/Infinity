O posicionamento em CSS é fundamental para criar layouts complexos.

A propriedade static é o comportamento padrão de todos os elementos.

O posicionamento relative ajusta o elemento em relação à sua posição original.

Já o absolute remove o elemento do fluxo normal e o posiciona em relação ao seu contêiner pai.

Com o posicionamento fixed, um elemento permanece fixo na tela, mesmo ao rolar a página.

E o posicionamento sticky é a combinação de relative e fixed, perfeito para menus de navegação.

O Box Model define o espaçamento e o tamanho de cada caixa na tela.
